# Random Quote Generator

## Project Overview

The Random Quote Generator is a simple and interactive web application that displays inspiring and motivational quotes randomly. The application allows users to generate new quotes with a single click and provides an enjoyable experience for users looking for daily motivation and inspiration.

The main objective of this project is to create a simple user-friendly application using frontend technologies and demonstrate the use of JavaScript for dynamic content generation.

## Features

* Generate random quotes
* Display different quotes on every click
* Simple and attractive user interface
* Dynamic content updates
* Responsive design
* Easy-to-use functionality

## Technologies Used

* HTML
* CSS
* JavaScript

## Working of the Application

The Random Quote Generator stores a collection of quotes in the application. When the user clicks the "Generate Quote" button, JavaScript randomly selects a quote from the collection and displays it on the screen. The quote changes every time the user requests a new one.

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open the index.html file in a web browser.
4. Click the "Generate Quote" button to view random quotes.
